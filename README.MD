## âœï¸ Kustomisasi

### Mengubah Warna

Edit variabel CSS di bagian `:root` pada file `assets/css/style.css`:

```css
:root {
    --forest-dark: #2d4a3e;
    --forest-medium: #3d5a4d;
    /* ... dan seterusnya */
}
```

### Menambah Section

Tambahkan section baru di `index.html` dan style-nya di `style.css`. Jangan lupa update navigation menu.

### Mengubah Skills

Edit bagian `.skills-grid` di `index.html` untuk menambah/mengurangi skill icons.
